ประวัติการศึกษา

ช่วงเวลา : พ.ศ. 2566 - ปัจจุบัน

สถานศึกษา : มหาวิทยาลัยธุรกิจบัณฑิตย์

ระดับการศึกษา : ปริญญาตรี

คณะ : วิทยาลัยวิศวกรรมศาสตร์และเทคโนโลยี

สาขา : วิศวกรรมคอมพิวเตอร์

ช่วงเวลา : พ.ศ. 2564 - 2566

สถานศึกษา : วิทยาลัยอาชีวศึกษาชลบุรี

ระดับการศึกษา : ประกาศนียบัตรวิชาชีพชั้นสูง ( ปวส. )

สาขา : เทคโนโลยีธุรกิจดิจิทัล

ช่วงเวลา : พ.ศ. 2561 - 2564

สถานศึกษา : วิทยาลัยอาชีวศึกษาชลบุรี

ระดับการศึกษา : ประกาศนียบัตรวิชาชีพ ( ปวช. )

สาขา : คอมพิวเตอร์ธุรกิจ

ช่วงเวลา : พ.ศ. 2558 - 2561

สถานศึกษา : โรงเรียนบ้านสวน(จั่นอนุสรณ์)

ระดับการศึกษา : มัธยมศึกษาตอนต้น

ช่วงเวลา : พ.ศ. 2549 - 2558

สถานศึกษา : โรงเรียนธรรมวาที

ระดับการศึกษา : อนุบาล - ประถมศึกษา
